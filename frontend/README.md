# AI Expense Tracker using MERN Stack and Gen AI

Welcome to the **AI Expense Tracker using MERN Stack and Gen AI Project build ** by **Abhishek Vishwakarma**.

This is an individual project by **Abhishek**, built as a modern **AI Expense Tracker and Financial Advisor Web Application** using MERN Stack and Gen AI integration from scratch.

This project is designed to help students, developers, and beginners learn real-world full stack web development with practical implementation, clean UI design, dashboard development, AI integration, authentication, charts, financial analytics, and backend API development.

## What We Will Build in This Series

- Modern responsive landing page
- Authentication system with login and register
- Professional dashboard UI
- Income and expense management
- Category-wise transactions
- Financial analytics charts
- Monthly reports system
- Saving goals management
- AI financial advisor
- AI chat assistant
- Budget planning system
- Money waste detection
- MERN Stack backend APIs
- MongoDB database integration
- Gen AI API integration
- Fully responsive design

## Frontend Setup

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Project Goal

The goal of this project is to create a practical finance management application where users can track income, manage expenses, analyze spending patterns, generate reports, plan budgets, and receive AI-powered financial guidance.
